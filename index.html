Question 1: You load a CSV file where the first five rows are metadata. How should you prepare it for use in Power BI?
Correct Answer: Remove Top Rows then Promote Headers
You must first remove the non-header metadata rows, then promote the correct row as headers.
Question 2: You are loading a folder of Excel files with similar structures into Power BI. What’s the best way to combine them?
Correct Answer: Use Combine Files in Power Query

Question 3: You need to create a new column that splits full names into first and last. What should you use?

Question 4: Which transformation ensures that Power BI recognizes dates correctly in a column?
Correct Answer: Change Type to Date

Question 5: You want to exclude rows where 'Status' is null. What’s the fastest step in Power Query?

Question 6: Which feature in Power BI enables combining columns from two tables based on a common key?
Correct Answer: Merge Queries
Merging is used to join data based on a shared column key (like SQL JOIN).
Question 7: You need to clean text values in a column with extra spaces. What’s your go-to step?

Question 8: Which step is typically needed before pivoting a column in Power Query?
Correct Answer: Unpivot Columns

Question 9: Which Power Query operation combines two tables with identical column structures?

Appending stacks data vertically when column names and types match.
